## Program Overview

The **NETSOL Institute of AI (NIAI)** offers a rigorous, industry-aligned AI/ML training program designed to bridge the gap between academic knowledge and real-world application. The program runs for **12 weeks (3 months)** and equips learners with both technical expertise and professional soft skills needed to thrive in AI-driven industries.

| Detail | Info |
|---|---|
| **Duration** | 12 Weeks (3 Months) |
| **Technical Hours** | 144 in-class credit hours |
| **Schedule** | 4 days/week (Technical) + 1 day/week (Soft Skills) |
| **Soft Skills Hours** | 24 hours of corporate readiness training |
| **Focus Areas** | Machine Learning, Deep Learning, NLP, Cloud AI (Azure/GCP/AWS), MLOps, Generative AI |

## AI/ML Technical Curriculum

The technical core consists of **144 in-class credit hours** delivered over **4 days per week**.

> **Note:** Module numbers reflect the original curriculum identifiers. Gaps in numbering (e.g., modules 7 and 10) indicate topics that are integrated into adjacent modules within this 12-week delivery schedule.

| Week(s) | Module | Key Topics & Units |
|---------|--------|-------------------|
| **1–2** | **1. Foundation** | Math for AI/ML (Linear Algebra, Calculus), Probability & Statistics, Python Programming (NumPy, Pandas, Scikit-learn) |
| **3** | **2. AI/ML & Cloud Platforms** | Real-world applications (Healthcare, Finance), AI Pipelines, and Cloud Ecosystems (GCP/AWS/Azure) |
| **4** | **3. Data Pipeline** | Data Preprocessing, Cleaning, Transformation, and ETL Processing |
| **5–6** | **4. Basic AI & ML** | Supervised Learning (Regression, Classification), Decision Trees, and Neural Network Foundations |
| **6–7** | **5. Advanced AI & ML** | Deep Learning (CNNs, RNNs, Transformers), Optimization, and Cloud-based Training on TPUs/GPUs |
| **8** | **6. Unsupervised Learning** | Clustering (K-Means, DBSCAN) and Dimensionality Reduction |
| **9** | **8. ML Pipeline & Automation** | MLOps Principles, CI/CD, and Workflow Automation (Kubeflow) |
| **10** | **9. Model Optimization** | Pruning, Quantization, and Performance Monitoring with Vertex AI |
| **11** | **11. Responsible AI** | Bias Detection, Explainability (SHAP, LIME), and Ethical AI Frameworks |
| **12** | **12. Generative AI** | Basics, Technologies, and Industrial Applications |
| **9–12** | **13. Capstone Project** | Real-world solutions (e.g., Medical Diagnostics, Fraud Detection) — runs in parallel with Weeks 9–12 |

## Soft Skills & Leadership Modules

In addition to technical training, the program includes **24 hours of corporate readiness training** (1 day per week), covering:

### Module 1 — Building Future-Ready Tech Leaders
- Leadership personal branding
- Agility and adaptive leadership in tech environments

### Module 2 — Teamwork & Collaboration
- Conflict resolution strategies
- Lencioni's 5 Dysfunctions of a Team framework

### Module 3 — Emotional Intelligence & Empathy
- Goleman's Emotional Intelligence Model
- Social awareness and relationship management in professional settings

### Module 4 — Communication Improvement
- Presentation and public speaking skills
- Body language and non-verbal communication
- Written fluency for professional and technical contexts

## Getting Started

### Prerequisites

```bash
pip install numpy pandas scikit-learn matplotlib pillow jupyter
```

### Running the Notebooks

1. Clone the repository:
   ```bash
   git clone https://github.com/safwanahmadsaffi/ai-ml-netsol.git
   cd ai-ml-netsol
   ```

2. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

3. Open any notebook from the `Python - Basics/` or `Exploratory Data Analysis - EDA/` folders and run the cells.
